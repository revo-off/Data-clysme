# Dataclysme Visualisation (Simple)

This dashboard is based on the API datamarts and provides 3 charts.

## Run locally

```bash
cd viz
pip install -r requirements.txt
streamlit run app.py
```

Then open http://localhost:8501

## What it shows

- Graph 1: Avg temperature by month (risks)
- Graph 2: Total precipitation by month (tourism)
- Graph 3: Avg temperature distribution (agriculture)
